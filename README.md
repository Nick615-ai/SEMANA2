Infografía Interactiva: Top 5 Tendencias en Educación
HTML5CSS3

📝 Descripción del Proyecto
Infografía interactiva de tipo lista ("Top 5") que presenta hallazgos clave basados en estudios científicos recientes sobre educación, inclusión, metacreatividad y habilidades cognitivas.

El proyecto transforma las viñetas tradicionales en iconos con un efecto de expansión (de semicírculo a círculo completo) e incorpora un resaltado animado con una línea ondulada bajo los títulos cuando el usuario pasa el cursor sobre cada elemento. Todo esto desarrollado de forma pura con HTML y CSS, sin dependencias de JavaScript.

🛠️ Tecnologías Utilizadas
HTML5: Estructura semántica (<ol>, <li>, <header>, <main>, <footer>).
CSS3:
Flexbox para el layout de los elementos.
Pseudo-elementos (::before, ::after) y border-radius para la animación de los iconos.
Imágenes SVG en base64 como background-image para la línea ondulada.
@keyframes y transition para las animaciones.
Diseño responsive bajo la filosofía Mobile First.
🚀 Cómo visualizar el proyecto
Este proyecto es estático, por lo que no requiere instalación de dependencias ni servidores locales.

Descarga o clona el repositorio en tu equipo.
Navega a la carpeta raíz del proyecto.
Haz doble clic sobre el archivo index.html.
El archivo se abrirá automáticamente en tu navegador web predeterminado (Chrome, Firefox, Edge, Safari, etc.).
